# Stock Trader

Project from [VueJS 2 Udemy Course](https://www.udemy.com/vuejs-2-the-complete-guide)

## Build Setup

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:8080
$ yarn run dev

# build for production with minification
$ yarn run build
```
